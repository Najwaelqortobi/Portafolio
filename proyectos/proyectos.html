Proyecto Arco

Este proyecto responde a la necesidad del Club Orion Getxo de contar con presencia en internet para darse a conocer, poder mostrar su actividad (reuniones, quedadas, campeonatos, etc.). También para poder captar asociados y darles toda la información necesaria del día a día del club.

En sste proyecto se ha realizado usando lenguajes de programación HTML5 y CSS3 en los que el equipo de Grupo3 tiene sobrada experiencia, mediante el desarrollo estructurado de las páginas web siguiendo un estándar de carpetas con documentos CSS propios de cada página web y un CSS general donde se recogen todo el diseño que debe ser común para el sitio.

Parseo y Casteo

Este proyecto se centra en la implementación de conceptos fundamentales de programación en Javascript, específicamente en el uso de parseo y casteo. La aplicación incluye herramientas prácticas como:

Calculadora de Préstamos: Esta herramienta permite a los usuarios calcular los pagos mensuales de un préstamo.

Generador de Personas: Esta aplicación genera perfiles aleatorios de personas, incluyendo atributos como nombre, edad y ocupación.
