Proyecto Bisuteria

Este proyecto consiste en una página web de una tienda de bisutería online, desarrollada con HTML5 CSS3 y JavaScript, que ofrece un diseño moderno y totalmente responsivo.

La plataforma integra un catálogo dinámico de productos mediante objetos JSON, un sistema de cuentas de usuario y un proceso de pedidos intuitivo, proporcionando una experiencia de compra fluida y atractiva que permite a los clientes explorar, seleccionar y adquirir bisutería de manera sencilla y elegante.

Parseo y Casteo

Este proyecto se centra en la implementación de conceptos fundamentales de programación en Javascript, específicamente en el uso de parseo y casteo. La aplicación incluye herramientas prácticas como:

Calculadora de Préstamos: Esta herramienta permite a los usuarios calcular los pagos mensuales de un préstamo.

Generador de Personas: Esta aplicación genera perfiles aleatorios de personas, incluyendo atributos como nombre, edad y ocupación.
